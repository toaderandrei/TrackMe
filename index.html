TrackMe

This is an application that tracks the user's path. It is based, to some extent to the MyTracks app from google. I tried to simplify it a bit, plus added my own database my own tracking data and some big changes to the way the service interacts with the UI. The user has the possibility to start a new track, running, walking, etc and record its route. Routes can be saved, if desired, to the database.
